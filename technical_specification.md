# Техническое задание: SimplePaint

## Архитектура проекта

### Модули программы:
1. main.py - главный файл для запуска приложения
2. ui/main_window.py - основное окно программы
3. ui/tools_panel.py - панель инструментов
4. ui/color_picker.py - выбор цвета
5. models/drawing_model.py - логика рисования
6. utils/file_operations.py - работа с файлами

### Пользователи программы:
Программа предназначена для одного пользователя. Все данные хранятся локально на компьютере.

### Хранение данных:
Пользовательские данные (рисунки) сохраняются в файлы форматов:
- PNG (.png)
- JPEG (.jpg, .jpeg)

Настройки программы (последний использованный цвет, размер кисти) будут сохраняться в простой текстовый файл settings.txt.

### Главное окно при запуске:
При запуске программы пользователь видит:
- Белое рабочее поле (холст) для рисования
- Меню сверху: "Файл", "Правка", "Справка"
- Панель инструментов слева или сверху с кнопками:
  - Кисть
  - Линия  
  - Прямоугольник
  - Эллипс
  - Ластик
- Панель цветов с основными цветами
- Слайдер для выбора размера кисти
- Строка состояния внизу с информацией о текущем инструменте

### Дополнительные окна:

- Простая информация о программе
- Кнопка "OK" для закрытия

#### Диалог сохранения файла
- Стандартный диалог сохранения файла

#### Диалог открытия файла

## Описание классов

### Класс MainWindow (главное окно)
- Создает и управляет всеми элементами интерфейса
- Обрабатывает меню и кнопки
- Содержит холст для рисования

### Класс DrawingCanvas (холст для рисования)
- Отвечает за отрисовку на холсте
- Обрабатывает события мыши (нажатие, перемещение, отпускание)
- Хранит текущее изображение

### Класс Tool (базовый класс инструментов)
- Абстрактный класс для всех инструментов рисования
- Методы: mouse_press, mouse_move, mouse_release

### Классы-наследники Tool:
- BrushTool - инструмент "Кисть"
- LineTool - инструмент "Линия"
- RectangleTool - инструмент "Прямоугольник"
- EllipseTool - инструмент "Эллипс"
- EraserTool - инструмент "Ластик"

### Класс ColorPicker (выбор цвета)
- Палитра основных цветов
- Возможность выбора произвольного цвета

### Класс FileManager (работа с файлами)
- Сохранение изображения в файл
- Загрузка изображения из файла

## Процесс работы пользователя

### Начало работы:
1. Пользователь запускает программу
2. Открывается главное окно с чистым холстом
3. Пользователь выбирает инструмент и цвет
4. Начинает рисовать на холсте

### Во время работы:
- Пользователь может менять инструменты
- Менять цвет и размер кисти
- Исправлять ошибки ластиком
- Сохранять промежуточные результаты

### Завершение работы:
1. Пользователь сохраняет готовый рисунок
2. Закрывает программу через меню "Файл → Выход" или крестик

## Технические требования

### Используемые технологии:
- Python 3.8+
- PyQt6 для графического интерфейса
- Стандартные модули Python для работы с файлами

### Требования к системе:
- Windows 10/11 или Linux
- Минимум 2 ГБ оперативной памяти
- 100 МБ свободного места на диске
